# AI Agent Assignments

Suggested usage:

## Opus / strong reasoning model

- architecture review
- security review
- domain boundary review
- complex design decisions

## Codex / coding agent

- implement a single story
- add tests
- modify code under strict repo docs

## Fabel / exploratory tool

- idea exploration
- UI variations
- non-authoritative prototypes

Do not use exploratory outputs as production code without review.
