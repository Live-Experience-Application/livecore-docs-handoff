# Repo Governance

## Repository owners

Recommended:

| Repo | Owner |
|---|---|
| livecore-platform | Core/platform owner |
| arcanos-app | Product/vertical owner |
| scenarioos-enterprise | Enterprise owner |
| livecore-deploy | Ops owner |
| livecore-docs-handoff | Architecture/product owner |

Initially, you may be all owners.

## Branch policy

- protect main
- require PR
- require tests once available
- no direct push to main after initial docs commit

## Issue policy

Every implementation task must link to a story ID.
